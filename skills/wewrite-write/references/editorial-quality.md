# 编辑质量标准

五项各按 1-5 分判断。交付要求是平均分至少 4，单项不低于 3，且没有阻断问题。

1. **准确**：核心事实有直接来源；事实、推断、意见分得开；结论没有超出证据边界。
2. **观点**：核心判断明确，说明新增价值、适用范围和最强反方，不重复常识充数。
3. **有用**：真正回答目标读者的问题，给出可执行行动、决策条件或可迁移的理解框架。
4. **合声**：符合账号语气但不被人格模板绑架；个人经历只来自本次用户材料；没有挪用范文。
5. **好读**：开头迅速进入问题，各节各有任务，论证连贯，标题准确，语言直接且无重复。

以下任一项出现都不能交付：

- 编造作者亲历、朋友、采访、对话、场景或身份。
- 核心具体事实没有来源，或来源不支持正文说法。
- 核心论点与证据脱节，或把推断写成确定事实。
- 文章没有给目标读者明确的结论、行动或判断方法。

## 编辑决定

- `pass`：没有阻断问题，五项达到交付要求。
- `revise`：问题能在现有材料内修正。必须修改成稿并重新检查，不能只列建议。
- `needs_input`：只有当用户明确要求写其个人故事，而关键材料缺失且无法安全换框架时使用；
  其他情况自动缩小主张或换框架。

最多两轮。第二轮仍未达到要求时，不得假装通过：删除不可靠内容、缩小承诺，交付能被证据
支持的版本。语言风险工具只能定位套话、碎句和节奏过齐，不能替代编辑判断，也不能为了
分数制造错句、情绪配额或随机离题。
